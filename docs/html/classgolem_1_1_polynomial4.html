Detailed Description

3rd-degree polynomial trajectory.

f(t) = a3*t3 + a2*t2 + a1*t + a0

A polynomial f() within a single window can be unambiguously specified by positions (p0, p1) and velocities (v0, v1) on its boundaries at the time t0 and t1 suitably, i.e. there are 4 equations:

f(t0) = p0

f'(t0) = v0

f(t1) = p1

f'(t1) = v1
