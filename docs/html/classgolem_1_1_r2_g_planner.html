R2GPlanner. Basic class for planning Reach-and-grasp trajectory. This class provides (1) a sampling method to generate a low dimension representation of uncertainty over object poses, and (2) a planner which builds trajectories that are more likely to distinguish between the miximum likelihood pose of the object and any sampled poses. Based on Platt R. et al. "A hypothesis-based algorithm for planning and control in non-Gaussian belief spaces", 2011.
