Detailed Description

Abstract class implementing a generic A* graph search algorithm.

A* graph search algorithm searches for the minimum cost path between root and goal nodes. By overriding pure abstract findCost(), nextNode() and other functions from protected part of the class, one can obtain eg. breath-first or depth-first search strategies, as well as different memory management strategies for finite or infinite (say very large) number of nodes. GraphSearch has a nearly optimal design - the only penalty (negligible in practise) are virtual function calls. GraphSearch uses some extra improvements like efficient node cost sorting.

See also
"Artificial Intelligence: A new synthesis", Nils J. Nilsson, 1998, Morgan Kaufmann
