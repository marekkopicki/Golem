Generates random rotation around (1, 0, 0, 0) on 4-sphere with dispersion parameter k (1/k is analogous to sigma^2), according to a von Mises-Fisher distribution. For k = 0 the distribution is uniform, for k >> 0 the distribution is concentrated around (1, 0, 0, 0). see A.T.A. Wood., "Simulation of the von-Mises Distribution", Communications of Statistics, Simulation and Computation, 23:157{164, 1994}
